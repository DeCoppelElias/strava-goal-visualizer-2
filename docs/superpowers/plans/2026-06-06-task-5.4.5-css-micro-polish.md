# TASK-5.4.5 — CSS Micro-Polish: Unit Label Color and Page Fade-In

**Goal:** Two small CSS improvements:
1. The `"km"` unit label next to the goal input uses `--text-3` (barely visible). `--text-2` is the correct choice for secondary UI labels.
2. The style guide specifies a page-level fade-in (`opacity 0→1, 200ms ease`) that is not yet implemented anywhere.

---

## File Map

| File | Action |
|------|--------|
| `frontend/src/index.css` | Modify — fix label colour; add `@keyframes` and apply to page roots |

---

### Step 1 — Fix `.goal-input__unit` colour

```css
/* Before */
.goal-input__unit {
  font-family: 'JetBrains Mono', monospace;
  font-size: 13px;
  color: var(--text-3);
}

/* After */
.goal-input__unit {
  font-family: 'JetBrains Mono', monospace;
  font-size: 13px;
  color: var(--text-2);
}
```

---

### Step 2 — Add `@keyframes fade-in`

Add near the top of the Motion section (or just before the Loading block at the bottom):

```css
@keyframes fade-in {
  from { opacity: 0; }
  to   { opacity: 1; }
}
```

---

### Step 3 — Apply to page roots

`.app-shell` is the authenticated page root. `.login-center` is the centred login content (not `.login-root`, which wraps the GDPR footer too — the fade should cover the visible content area, not the footer separately).

```css
/* In .app-shell */
.app-shell {
  display: flex;
  flex-direction: column;
  min-height: 100vh;
  animation: fade-in 200ms ease forwards;
}

/* In .login-center */
.login-center {
  flex: 1;
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  padding: 64px 32px;
  animation: fade-in 200ms ease forwards;
}
```

---

### Step 4 — Verify

- Goal input: `"km"` unit label is visibly readable in both dark and light mode.
- Authenticated shell: page fades in smoothly on first load / after login.
- Login page: login card fades in on load.
- No flicker or layout shift during animation.
- Theme toggle does NOT re-trigger the fade (it only fires on mount since `animation` is not in the `transition` list).

---

### Step 5 — Commit

```bash
git add frontend/src/index.css
git commit -m "fix(frontend): unit label uses --text-2; add 200ms page fade-in to app shell and login"
```
