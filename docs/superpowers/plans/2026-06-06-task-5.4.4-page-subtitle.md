# TASK-5.4.4 — Replace Raw Athlete ID in Page Subtitle

**Goal:** The page subtitle currently shows `Athlete #12345678` — a raw internal Strava ID that is opaque and meaningless to the user. Replace it with the current calendar year and dashboard context.

---

## File Map

| File | Action |
|------|--------|
| `frontend/src/pages/DashboardPage.tsx` | Modify — change subtitle content |

---

### Step 1 — Update the subtitle in `DashboardPage.tsx`

```tsx
// Before (line ~105)
<p className="page-subtitle">Athlete #{athleteId}</p>

// After
<p className="page-subtitle">{new Date().getFullYear()} · Running Goal</p>
```

The `athleteId` prop is still accepted by the component (the parent passes it for data-fetching purposes) and remains in the `Props` interface — it just no longer appears in the UI.
